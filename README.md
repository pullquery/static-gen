# Static Generator

Static Generator generate static HTML from Markdown.

## Directories
- src: TypeScript source
- dist: JavaScript app
- app
    - papers: Markdown articles
    - assets: Assets of articles
    - styles: CSS
    - scripts: Client-side JavaScript
    - templates: EJS templates
- static: HTML pages
